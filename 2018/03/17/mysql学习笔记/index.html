分为两种方式：手动安装版的和自动安装版的，其中的问题真的好多。。。

其中遇到的问题有：

报错信息有一句提示到：该应用程序需要Visual Studio 2013 Redistributable。请安装Redistributable并再次运行此安装程序。

找了一下，在微软的官网上可以[下载]到(https://link.jianshu.com/?t=https://www.microsoft.com/zh-cn/download/details.aspx?id=40784)

重新安装Mysql即可安装成功。

  • 其中还看出现的问题的安装界面一直卡在一个地方，解决方法是：

1、C:\Windows\system32>net start mysql
发生系统错误 2。

系统找不到指定的文件。

参见博客博客

2、关于卸载了重新安装的时候总是会卡住，mysql 安装到最后一步时,一直会失败会卡住的问题：

主要可以从以下几个方面解决：

  • 安装时最后一步出现服务器未响应的解决办法 ​​​​
    (1)在控制面板–>程序–>卸载mySQL
    (2)在相应盘下删除MySQL文件
    (3)C盘的programData目录下显示隐藏文件，将MySQL文件删除。
    (4)删除注册表中的三项： 
    HKEY_LOCAL_MACHINE/SYSTEM/ControlSet001/Services/Eventlog/Application/MySQL 
HKEY_LOCAL_MACHINE/SYSTEM/ControlSet002/Services/Eventlog/Application/MySQL 
HKEY_LOCAL_MACHINE/SYSTEM/CurrentControlSet/Services/Eventlog/Application/MySQL

有可能部分注册表项没有，将能找到的mysql相关注册表项全删除，可以通过搜索MySQL关键字删除注册表项; 

(5)重新安装MySQL就可以了。
